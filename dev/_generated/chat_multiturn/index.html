Here’s a simple example of multiple dispatch in Julia:

julia function area(shape::Tuple{Symbol, Float64}) if shape[1] == :circle return π * shape[2]^2 elseif shape[1] == :rectangle return shape[2] * shape[3] end end

Example usages:

circlearea = area(:circle, 5.0) # Area of a circle with radius 5 rectanglearea = area(:rectangle, 4.0, 6.0) # Area of a rectangle 4x6

println("Circle Area: ", circlearea) println("Rectangle Area: ", rectanglearea)


In this example, the `area` function behaves differently depending on the shape type (`:circle` or `:rectangle`) and its corresponding dimensions.
